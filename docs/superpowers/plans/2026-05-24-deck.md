# DeckPage Swipe Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the full DeckPage swipe experience — card stack orchestration, swipe persistence to Supabase, and end-of-deck screen — using the existing IdeaCard component untouched.

**Architecture:** DeckPage owns all deck state (`currentIndex`, `savedCount`, `isDone`). On each swipe it fire-and-forgets a Supabase insert, then advances the card index. When all 10 cards are swiped, it shows an end-of-deck screen with saved count and navigation CTAs.

**Tech Stack:** React 19, Framer Motion (via IdeaCard — untouched), Supabase JS v2, React Router v7, inline styles (Sapphire & Ruby palette)

```jsx
export default function DeckPage() {
  const { state } = useLocation()
  const navigate = useNavigate()
  const sessionRef = useRef(null)

  const [currentIndex, setCurrentIndex] = useState(0)
  const [savedCount, setSavedCount] = useState(0)
  const [isDone, setIsDone] = useState(false)

  const ideas = state?.ideas ?? []
  const city = state?.city ?? ''
  const time = state?.time ?? ''

  useEffect(() => {
    if (!ideas.length) {
      navigate('/input', { replace: true })
      return
    }
    supabase.auth.getSession().then(({ data: { session } }) => {
      sessionRef.current = session
    })
  }, [])

  function handleSwipe(direction) {
    const idea = ideas[currentIndex]

    if (sessionRef.current?.user) {
      supabase.from('swipes').insert({
        user_id: sessionRef.current.user.id,
        idea_id: idea.id,
        direction,
        idea_data: idea,
      })
    }

    if (direction === 'right') setSavedCount(n => n + 1)

    const next = currentIndex + 1
    if (next >= ideas.length) {
      setIsDone(true)
    } else {
      setCurrentIndex(next)
    }
  }

  if (isDone) {
    return (
      <div style={{ minHeight: '100dvh', background: C.page, display: 'flex', flexDirection: 'column', maxWidth: 430, margin: '0 auto' }}>
        <EndOfDeckScreen
          savedCount={savedCount}
          total={ideas.length}
          onViewMatches={() => navigate('/matches')}
          onShuffle={() => navigate('/input')}
        />
      </div>
    )
  }

  const contextLabel = [city, time].filter(Boolean).join(' · ')

  // Render up to 3 cards back-to-front (last in DOM = on top)
  const visibleIndices = [
    Math.min(currentIndex + 2, ideas.length - 1),
    Math.min(currentIndex + 1, ideas.length - 1),
    currentIndex,
  ].filter((v, i, arr) => arr.indexOf(v) === i)

  return (
    <div style={{ minHeight: '100dvh', background: C.page, display: 'flex', flexDirection: 'column', maxWidth: 430, margin: '0 auto' }}>

      {/* Top bar */}
      <div style={{ display: 'flex', justifyContent: 'space-between', alignItems: 'center', padding: '48px 22px 18px' }}>
        <button
          onClick={() => navigate('/input')}
          style={{ background: 'none', border: 'none', cursor: 'pointer', display: 'flex', alignItems: 'center', gap: 6, padding: 0 }}
        >
          <ChevronLeft />
          <span style={{ fontFamily: '"DM Mono", monospace', textTransform: 'uppercase', fontSize: 9, letterSpacing: '0.26em', color: C.inkDim }}>Back</span>
        </button>
        <div style={{ display: 'flex', alignItems: 'center', gap: 6 }}>
          <MarkSmall />
          <span style={{ fontFamily: '"Playfair Display", Georgia, serif', fontSize: 13, fontWeight: 500, color: C.ink }}>
            Out <span style={{ fontStyle: 'italic', color: C.wine }}>Tonight</span>
          </span>
        </div>
        {contextLabel ? (
          <span style={{ fontFamily: '"DM Mono", monospace', fontSize: 9, letterSpacing: '0.18em', color: C.inkDim, textAlign: 'right', maxWidth: 90 }}>
            {contextLabel}
          </span>
        ) : <div style={{ width: 60 }} />}
      </div>

      {/* Card stage */}
      <div style={{ flex: 1, display: 'flex', flexDirection: 'column', alignItems: 'center', justifyContent: 'center', paddingBottom: 6 }}>
        <div style={{ position: 'relative', width: 312, height: 440 }}>
          {visibleIndices.map((ideaIdx) => {
            const isActive = ideaIdx === currentIndex
            const stackOffset = ideaIdx - currentIndex
            return (
              <IdeaCard
                key={ideas[ideaIdx].id}
                idea={ideas[ideaIdx]}
                imageUrl={null}
                currentNum={ideaIdx + 1}
                total={ideas.length}
                onSwipe={isActive ? handleSwipe : undefined}
                isActive={isActive}
                stackOffset={stackOffset}
              />
            )
          })}
        </div>

        <ActionButtons
          onPass={() => handleSwipe('left')}
          onLike={() => handleSwipe('right')}
        />
      </div>

    </div>
  )
}
```

**Note:** Action button taps advance the card instantly (no swipe-off animation). Drag swipes animate off via IdeaCard's Framer Motion controls — this is correct and expected behavior.
