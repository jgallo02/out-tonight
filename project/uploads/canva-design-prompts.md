## PROMPT 1 — App Logo / Wordmark

**Use in:** Canva → Create a design → Logo

```
Design a wordmark logo for a date night discovery app. The name is Out Tonight.

Style: Warm, editorial, sophisticated but not stuffy. Think a luxury city magazine meets a modern dating app. Not corporate. Not cute. Grown-up and distinctive.

Typography direction: Pair an elegant serif (similar to Playfair Display or Cormorant Garamond) for the main word with a small geometric sans or monospace detail element. The serif should have an italic weight option — use it.

Color: Deep near-black background (#0d0d0d). Wordmark in warm cream (#f0ece4). Optional accent: a single amber spark or asterisk glyph in #c9803a.

Do NOT use: hearts, map pins, fork-and-knife icons, generic app badge shapes, gradients on the text itself.

Optional mark: A small abstract symbol — could be a star fragment (✦), a compass rose simplified to 4 points, or a single elegant flourish — used as an app icon standalone.

Deliver: Horizontal lockup (wordmark + optional mark left) AND stacked version (mark above wordmark). Both on dark background.
```

## PROMPT 3 — App Card UI Reference (Front)

**Use in:** Canva → Create a design → Mobile (375x812px / iPhone frame)

```
Design a single mobile UI card for a date night discovery app. This is a reference mockup for the core "swipe card" component.

Card dimensions: approximately 343x473px, centered in a dark phone screen (#0d0d0d background).

Card has:
- Full-bleed photographic background image (use a beautiful photo of a jazz bar or intimate restaurant — dark, candlelit, atmospheric)
- Heavy dark gradient overlay rising from the bottom (transparent at top, 95% black at bottom)
- Subtle drop shadow: 0 32px 80px rgba(0,0,0,0.7)
- Rounded corners: 20px

On top of the gradient, bottom-anchored content:
- Category badge: small pill, amber border and text, monospace font, uppercase — "NIGHTLIFE"
- Title: large italic serif, warm cream — "Jazz Supper at Mezzrow" (2 lines max)
- Tagline: light sans, muted cream — "The city's most intimate jazz room, 40 seats only"
- Tag row: 3 small outlined pills — "Romantic" · "Formal" · "Cultural"
- Meta row: amber monospace "$$$" + small text "~$110/person" + right-aligned "3 hrs"
- Very bottom: tiny uppercase monospace hint "TAP TO FLIP" in near-invisible muted

No visible borders on the card edges — let the drop shadow define it.

Show two action buttons below the card: a round ✕ button (left, dark red tint) and a round ❤ button (right, dark green tint). Both slightly frosted glass feel.
```

## PROMPT 4 — App Card UI Reference (Back / Flipped)

**Use in:** Canva → Create a design → Mobile (375x812px)

```
Design the "flipped back" state of a mobile swipe card for a date night app. Same card dimensions as front (343x473px, 20px radius, centered on #0d0d0d).

Card back has:
- Solid dark background: linear gradient from #1a1512 to #0f0d0b
- Subtle amber hairline at very top center (a 1px gradient line, transparent → amber → transparent)

- No photo

Content (top to bottom, left-aligned, 24px padding):
- Small monospace uppercase amber label: "NIGHTLIFE"

- Body text paragraph (3 sentences, light sans, muted cream, 13pt): description of the experience
- "WHY TONIGHT" callout block: left border 3px amber, background amber at 10% opacity, inside: tiny uppercase label "WHY TONIGHT" in amber, then italic sans body text below it
- "GETTING THERE" info block: very dark surface, tiny uppercase muted label, then logistics text in muted cream
- Very bottom centered: tiny "TAP TO FLIP BACK" in near-invisible muted monospace

The back should feel like reading a beautifully typeset card insert — editorial, calm, informative.
```

## PROMPT 5 — Splash / Login Screen

**Use in:** Canva → Create a design → Mobile (375x812px)

```
Design a full-screen mobile splash/login screen for a date night discovery app called Out Tonight.

Background: Deep dark (#0d0d0d) with a very subtle warm radial glow at center (amber at 4% opacity fading out). Add an extremely subtle noise/grain texture overlay at 3% opacity.

Layout (centered vertically, slight upward bias):
- App wordmark / name at top of center — italic serif, warm cream, large (40–48pt)
- Tagline directly below: "Tonight, make it worth remembering." — light sans, muted cream, 15pt, letter-spaced slightly
- Thin amber gradient divider line (full width, very faint)
- Generous space
- "Sign in with Google" button: rounded pill shape, white background, Google "G" logo on left, "Continue with Google" in dark text, 16pt. Full-width minus 40px margin.
- Below button: tiny muted text "By signing in you agree to our Terms & Privacy"

Bottom: A very faint, blurred partial photo peeking from the absolute bottom edge — suggesting the experience cards below. Nearly invisible. Creates depth and intrigue.

Do NOT add decorative hearts, sparkles, cliché couple silhouettes, or stock-photo hero images.
```

## PROMPT 6 — Input Form Screen

**Use in:** Canva → Create a design → Mobile (375x812px)

```
Design a mobile input/preferences form screen for a date night discovery app. Dark aesthetic matching established brand.

Background: #0d0d0d. Top: app wordmark small, left-aligned, with a "← Back" or close affordance right.

Section header: "Where are we going?" — italic serif, warm cream, 22pt

Form fields (show as designed UI components, not wireframes):
1. City input — dark surface pill with placeholder "New York, NY"
2. Date + Time row — two side-by-side dark surface pills: date on left, segmented "Evening" selector on right
3. "Who's coming?" — +/- stepper showing "2 people"
4. Vibe section — labeled "Tonight's vibe" then 10 pill chips in 3 rows: Romantic, Adventurous, Relaxed, Formal, Casual, Quirky, Cultural, Active, Cozy, Spontaneous. Selected chips: amber background + cream text. Unselected: dark surface + muted text
5. Budget slider — horizontal slider track, amber fill to left of thumb, label above: "$85 per person"
6. Food toggle — "Include food?" label with a toggle switch (amber when on)

Bottom: Full-width CTA button, amber-to-plum gradient, cream text "Find Ideas →", 16pt medium, 16px radius.

All form elements: dark surface (#1a1a1a), 1px border (#2a2a2a), 12px radius, generous padding.
```

## WHAT TO DO WITH THESE OUTPUTS

Once you've generated and exported these Canva screens:

1. **Export each as PNG** at 2x resolution
2. **Upload to a shared folder** (Google Drive or Dropbox)
3. **Add to your Lovable follow-up prompt** with language like:
   > "Use the attached screenshots as the exact visual reference for this app. Match colors, typography, card layout, and component styles precisely. Do not default to your own design system."
4. **Reference specific screens** per Lovable build step:
   - Auth: use Splash screen reference
   - Input: use Form screen reference
   - Deck: use Card Front + Card Back references
   - Matches: use Matches screen reference
